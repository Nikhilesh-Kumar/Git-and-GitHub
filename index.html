Welcome to Git and GitHub

Install git onto your system

Follow the Steps

Checking Installation:

After installing the git into your system check through the below command whether git is installed onto the system properly or not

  • git --version

    • Initializing Git:

    After confirming the git onto the system the git is initialized on the current project through below command

  • git init

    • Checking Status:

    After initializing git locally onto the system the status for the current untracked changes is checked using

  • git status

    • Adding Files:

    Now the untracked files can be added to the git using below command

  • git add filename (for a specific file)
  • git add . (for all untracked changes in the current directory)

    • You can also use

  • git add -A
    • to include deletions as well as new/modified files.

    Save the changes:

    To actually save these changes to your history after adding them, you need to commit them:

  • git commit -m "Your preferred message for the current save"

    • Confirm your Identity:

    Identify your self if you get error while doing commit for the changes

  • git config --global user.name "your name"
  • git config --global user.email "your email"

    • History of Committed Changes:

    For viewing the history of all committed changes or saved files in repository, use

  • git log

    • Switch to different version:

    If want to undo the saved changes and move to previous version of your code

  • git reset --hard "required previous version to switch"

    • If want to move to previous version of code without deleting the code

  • git reset --soft "required previous version to switch"

    • Create Repository in GitHub:

    After creating repository in GitHub use below 3 commands to push your code finally to GitHub

  • git remote add origin git@github.com:Nikhilesh-Kumar/Git-and-GitHub.git
  • git branch -M main
  • git push -u origin main
